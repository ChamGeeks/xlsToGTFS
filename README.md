# Chamonix bus excel to GTFS

Convert bus timetables in excel to GTFS (Work In Progress)

### Requirements
* Node.js > 7.6 
*Suggestion: [nvm](https://github.com/creationix/nvm)*

### Install
* `git clone git@github.com:ChamGeeks/xlsToGTFS.git`
* `cd xlsToGTFS`
* `npm install` or `yarn`
* `npm start` or `yarn start`
